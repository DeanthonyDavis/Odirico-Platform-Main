Connections and setup
Surge can sync Gmail and Google Calendar directly from your browser when it is running on localhost or a secure HTTPS origin and has your Google OAuth web client ID.
Google setup uses the browser token flow. Your OAuth client must be a Web application with every origin you plan to use listed as an Authorized JavaScript origin, including localhost for testing and your secure production domain.
